# 自己动手构建vue+electron项目
```
1. vue create my-project
2. npm install electron -D // 安装最新版本的electron
3. vue add electron-builder
```
因为如果直接执行第三行，会导致 electron 安装的不是最新版本，所以需要在执行前先在项目中安装好 electron 。
